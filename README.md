# NZone - Youth Sports Partnership Program

Youth sports program designed to partner with childcare centers (Kindercare, etc.) to provide after-school programming and generate parent signups.

## Project Structure

- **KINDERCARE_RESEARCH_PLAN.md** - Research strategy and phases for finding decision makers
- **kindercare_contacts.csv** - Contact database (created during research)
- **docs/** - Documentation and templates
- **CONTRIBUTING.md** - Workflow and contribution guidelines

## Quick Start

See [CONTRIBUTING.md](CONTRIBUTING.md) for the development workflow and how to contribute.

## Research Phases

1. **Phase 1:** Identify Kindercare locations
2. **Phase 2:** Find Center Directors and decision makers
3. **Phase 3:** Collect contact information
4. **Phase 4:** Create outreach templates and strategy

See `KINDERCARE_RESEARCH_PLAN.md` for detailed information.
