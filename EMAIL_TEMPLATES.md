## Email Best Practices

**Timing:**
- Send Tuesday-Thursday mornings (9am-11am local time)
- Avoid Mondays and Fridays
- Wait 5-7 days before first follow-up
- Wait another 5-7 days before second follow-up

**Tips:**
- Personalize the director name if possible
- Keep it short (under 150 words for first email)
- One clear call-to-action (request a call)
- Include your phone number - shows you're confident
- Follow up multiple times (different angles)

**Tracking:**
- Note date sent
- Track responses
- Update status in contact database
- Note any objections or questions

## Phone Call Script

If you reach them by phone:

**Opening:**
"Hi [Name], this is [Your Name] from Nzone Sports. I'm reaching out to directors in the Atlanta area about a partnership opportunity. Do you have 60 seconds?"

**Pitch:**
"We run youth sports leagues - soccer, flag football, basketball, cheerleading - one day a week. Parents sign their kids up and we handle everything. It's a way to add value for your families without any work from you. Have you heard of us?"

**Listen** - Let them respond

**Close:**
- If interested: "Great! Can I send you some info and set up a quick call?"
- If not interested: "No problem! Can I follow up in a few weeks?"
- If wrong person: "Who would be the right person to talk to about new programming?"
